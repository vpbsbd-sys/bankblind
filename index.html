🏦 About This Application

This is a self-service slip generation tool for Kolhapur Co-operative Bank members. It helps you generate transaction request slips which you present at the bank counter.

🔒 Data Privacy

  • Your registration details are saved only on your device (mobile phone) using browser local storage.
  • No personal data is sent to any external server.
  • No data is shared with any third party.
  • You can delete your saved data anytime using the "Clear My Data" option.

🔐 QR Code Security

  • QR codes are encrypted using AES-256 military-grade encryption.
  • Only the bank's authorised kiosk can read QR data.
  • Each QR code is valid for 48 hours only.
  • QR codes contain a tamper-detection checksum.
  • Keep your QR code private — do not share screenshots.

⚠️ Important Disclaimer

This app generates request slips only. It does not access your bank account, does not process transactions, and is not connected to any banking system. All transactions must be verified and processed by authorised bank staff.

📜 Compliance

This system is designed for internal slip pre-filling only. No sensitive financial data (PAN, Aadhaar, passwords, PINs) should be entered in this application.

🗑️

Clear My Data

This will permanently delete your saved registration details from this device. You will need to register again.

✅ This does NOT affect your bank account in any way.
